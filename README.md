# unifi-hysteria

Hysteria for UniFi OS. A powerful, lightning fast and censorship resistant proxy.

---

## About

This repository provides **Hysteria** packages specifically tailored for **UniFi OS**.

## Upstream

This project is a distribution of the original Hysteria project.

- **Source**: [apernet/hysteria](https://github.com/apernet/hysteria)
- **Official Website**: [hysteria.network](https://hysteria.network)

## Installation

1. Download the latest `.deb` package from the [Releases](https://github.com/IanSun/unifi-hysteria/releases) page.
2. Transfer the file to your UniFi device (via SCP or direct download).
3. Install using `dpkg`:

```sh
dpkg -i unifi-hysteria_*.deb
```
